# play-run-or-die
Welcome to unpack the PCK file. This project allows modification of the PCK content. If you wish to make changes, you can apply for fusion at "https://github.com/nm943snm-debug/play-run-or-die/issues" and provide a downloadable cloud drive URL (it must be a PCK made with GODOT 3.5 or the entire project's ZIP). I will then create a module and add a module loading section.

Note: If you are unfamiliar with GODOT 3.5, please email "nm943snm@gmail.com" and I will assist you in creating the module.
